两类问题（平面应力与平面应变）

问题 平面应力问题 平面应变问题
问题模型 等厚度薄板 等截面的长柱体
受力 平等于板面，且不沿厚度变化的面力或约束 平等于板面，且不沿长度变化的面力或约束
定量分析 $\sigma_{z}=0,\tau_{zx}=0,\tau_{zy}=0$（应力、应变、位移均是$x,y$的函数） $\tau_{zx}=0,\tau_{zy}=0$（应力、应变、位移均是$x,y$的函数）

一点的应力状态

[[材料力学#一点的应力状态]]
方向余弦定义为：$$
\begin{cases}
\cos(N,x)=l\ \cos(N,y)=m
\end{cases}
$$
任一方向的 $x,y$ 平等的应力： $$
\begin{cases}
p_{x}=l\sigma_{x}+m\tau_{yx}\
p_{y}=m\sigma_{y}+l\tau_{xy}
\end{cases}
$$
则，正应力与切应力： $$
\begin{cases}
\sigma_{N}=l^2\sigma_{x}+m^2\sigma_{y}+2lm\tau_{xy}\
\tau_{N}=lm(\sigma_{y}-\sigma_{x})+(l^2-m^2)\tau_{xy}
\end{cases}
$$
主应力方向(两个主应力方向相互垂直)上： $$
\sigma=\dfrac{\sigma_{x}+\sigma_{y}}{2}\pm \sqrt{ (\dfrac{\sigma_{x}-\sigma_{y}}{2})^2 +\tau_{xy}^2}
$$
第一应力不变量： $$
\sigma_{1}+\sigma_{2}=\sigma_{x}+\sigma_{y}
$$
最大剪应力发生在与主应力夹角 $45^。$方向上，为 $\pm\dfrac{\sigma_{1}-\sigma_{2}}{2}$

平衡微分方程

由于连续性，可以将不同变量的改变量用泰勒级数展开： $$
\sigma_{x+\Delta x}=\sigma_{x}+\dfrac{ \partial \sigma }{ \partial x } \mathrm{d}x+\dfrac{1}{2!}\dfrac{ \partial^{2} \sigma_{x} }{ \partial x^{2} } \mathrm{d}x^2+\cdots
$$
略去高阶得： $$
\sigma_{x+\Delta x}=\sigma_{x}+\dfrac{ \partial \sigma }{ \partial x } \mathrm{d}x
$$
对于其他应力相同，可以==证明切应力互等==
可以推出： $$
\begin{cases}
\dfrac{ \partial \sigma_{x} }{ \partial x } +\dfrac{ \partial \tau_{yx} }{ \partial y } +f_{x}=0\
\dfrac{ \partial \sigma_{y} }{ \partial y } +\dfrac{ \partial \tau_{xy} }{ \partial x } +f_{y}=0
\end{cases}

$$
几点注意：

  • $z$方向算成平衡
  • 与材料无关
  • 平面问题适用
  • 基本假设有连续性、均匀性、小变形

几何方程

$$
\begin{cases}
\varepsilon_{x}=\dfrac{ \partial u }{ \partial x } \
\varepsilon_{y}=\dfrac{ \partial v }{ \partial y } \
\gamma_{xy}=\dfrac{ \partial u }{ \partial x } +\dfrac{ \partial v }{ \partial y }

\end{cases}
$$

  • 用两个平行于 $x,y$轴的微小线段表示伸缩与旋转
  • 用到假设：连续性，均匀性，小变形
    应变$\nrightarrow$位移，位移$\rightarrow$应变
    令上方程组为0，得： $$
    u=f_{1}(y),v=f_{2}(x)
    $$
    由此，可以得到刚体位移（平移+转动）。

物理方程

广义胡克定律
[[材料力学#广义胡克定律]]
三向应力状态（一般情况）下：
$$\begin{cases}
\varepsilon_x=\dfrac{1}{E}[\sigma_x-\nu(\sigma_y+\sigma_z)]\
\varepsilon_y=\dfrac{1}{E}[\sigma_y-\nu(\sigma_z+\sigma_x)]\
\varepsilon_z=\dfrac{1}{E}[\sigma_z-\nu(\sigma_x+\sigma_y)]\
\gamma_{xy}=\dfrac{\tau_{xy}}{G}\
\gamma_{xz}=\dfrac{\tau_{xz}}{G}\
\gamma_{yz}=\dfrac{\tau_{yz}}{G}\
\end{cases}$$
$$
G=\dfrac{E}{2(1+\nu)}
$$

  • 用到假设：完全弹性，均匀性，各向同性，连续性，小变形

平面应力下：
$$\begin{cases}
\varepsilon_x=\dfrac{1}{E}(\sigma_x-\nu\sigma_y)\
\varepsilon_y=\dfrac{1}{E}(\sigma_y-\nu\sigma_z)\
\varepsilon_z=-\dfrac{\nu}{E}(\sigma_x+\sigma_y)\
\gamma_{xy}=\dfrac{\tau_{xy}}{G}\
\end{cases}$$
$$
\varepsilon_{z}=-\dfrac{\nu}{E}(\sigma_{x}+\sigma_{y})
$$
上式一般不为0，会产生底面畸变。

平面应变下：
$$
\begin{cases}
\varepsilon_{x}=\dfrac{1+\mu^2}{E}(\sigma_{x}-\dfrac{\mu}{1-\mu}\sigma_{y})\
\varepsilon_{y}=\dfrac{1+\mu^2}{E}(\sigma_{y}-\dfrac{\mu}{1-\mu}\sigma_{x})\
\gamma_{xy}=\dfrac{2(1+\mu)}{E}\tau_{xy}
\end{cases}

$$

边界条件

  • 精确边界条件为2个，模糊边界条件（圣维南原理）3个
    位移边界条件：体现位移保持连续性
    应力边界条件：体现平衡微分方程
    混合边界条件
  • 圣维南原理的前提：次要边界上、应力等效
  • 力矩方向：应力正方向与正力臂的乘积

位移法求解

平面应力问题
由物理方程反求应力：
$$
\begin{cases}
\sigma_{x}=\dfrac{E}{1-\mu^2}(\varepsilon_{x}+\mu\varepsilon_{y})\
\sigma_{y}=\dfrac{E}{1-\mu^2}(\varepsilon_{y}+\mu\varepsilon_{x})\
\tau_{xy}=\dfrac{E}{2(1+\mu)}\gamma_{xy}
\end{cases}
$$
可以推出： $$
\begin{cases}
\sigma_{x}=\dfrac{E}{1-\mu^2}(\dfrac{ \partial u }{ \partial x } +\mu \dfrac{ \partial v }{ \partial y } )\
\sigma_{y}=\dfrac{E}{1-\mu^2}(\dfrac{ \partial v }{ \partial y }+\mu\dfrac{ \partial u }{ \partial x })\
\tau_{xy}=\dfrac{E}{2(1+\mu)}(\dfrac{ \partial u }{ \partial x }+\dfrac{ \partial v }{ \partial y })
\end{cases}
$$
代入平衡微分方程：
$$
\begin{cases}
\dfrac{E}{1-\mu^2}(\dfrac{ \partial^{2} u }{ \partial x^{2} } +\dfrac{1-\mu}{2}\dfrac{ \partial^{2} u }{ \partial y^{2} } +\dfrac{1+\mu}{2}\dfrac{ \partial^{2} v }{ \partial x\partial y } )+f_{x}=0 \
\dfrac{E}{1-\mu^2}(\dfrac{ \partial^{2} v }{ \partial y^{2} } +\dfrac{1-\mu}{2}\dfrac{ \partial^{2} v }{ \partial x^{2} } +\dfrac{1+\mu}{2}\dfrac{ \partial^{2} u }{ \partial x\partial y } )+f_{y}=0 \
\end{cases}
$$
另外，边界条件可以写为：
$$
\begin{cases}
\dfrac{E}{1-\mu^2}[l(\dfrac{ \partial u }{ \partial x } +\mu\dfrac{ \partial v }{ \partial y } ){s}+m\dfrac{1-\mu}{2}(\dfrac{ \partial u }{ \partial y } +\dfrac{ \partial v }{ \partial x } )s ]=\bar{f{x}}\
\dfrac{E}{1-\mu^2}[m(\dfrac{ \partial v }{ \partial y } +\mu\dfrac{ \partial u }{ \partial x } ){s}+l\dfrac{1-\mu}{2}(\dfrac{ \partial v }{ \partial x } +\dfrac{ \partial u }{ \partial y } ){s} ]=\bar{f{y}} \
\end{cases}
$$

平面应变问题只需要进行材料的参数代换

力法求解

平面应力问题
相容方程(变形协调)（几何方程）：
$$
\dfrac{ \partial^{2} \varepsilon_{x} }{ \partial y^{2} } +\dfrac{ \partial^{2} \varepsilon_{y} }{ \partial x^{2} } =\dfrac{ \partial^2 \gamma_{xy} }{ \partial x \partial y }
$$
物理方程：
$$
\dfrac{ \partial ^2}{ \partial y^2 }(\sigma_{x}-\mu\sigma_{y})+ \dfrac{ \partial ^2}{ \partial x^2 }(\sigma_{y}-\mu\sigma_{x})=2(1+\mu)\dfrac{ \partial^{2} \tau_{xy} }{ \partial x\partial y}
$$
平衡微分方程：
$$
2\dfrac{ \partial^{2} \tau_{xy} }{ \partial x\partial y}=-\dfrac{ \partial^{2} \sigma_{x} }{ \partial x^{2} } -\dfrac{ \partial^{2} \sigma_{y} }{ \partial y^{2} } -\dfrac{ \partial f_{x} }{ \partial x } -\dfrac{ \partial f_{y} }{ \partial y }

$$
代入得：
$$
(\dfrac{ \partial^{2} }{ \partial x^{2} }+\dfrac{ \partial }{ \partial y } )(\sigma_{x}+\sigma_{y})=-(1+\mu)(\dfrac{ \partial f_{x} }{ \partial x } +\dfrac{ \partial f_{y} }{ \partial y } )
$$
平面应变问题只需要进行材料的参数代换

校核：

  • 平衡微分方程
  • 相容方程
  • ==边界条件==
  • 单连体可以得到确定解

常体力情况化简

相容方程可化为：
$$
(\dfrac{ \partial^{2} }{ \partial x^{2} }+\dfrac{ \partial^{2} }{ \partial y^{2} } )(\sigma_{x}+\sigma_{y})=0
$$
即： $$
\nabla(\sigma_{x}+\sigma_{y})=0
$$
==与材料无关==

应力函数

==常体力条件下==
$$
\nabla^4\Phi=0
$$
可以得到： $$
\begin{cases}
\sigma_{x}=\dfrac{ \partial^{2} \Phi }{ \partial y^{2} } \
\sigma_{y}=\dfrac{ \partial^{2} \Phi }{ \partial x^{2} } \
\tau_{xy}=-\dfrac{ \partial^{2} \Phi }{ \partial x\partial y }
\end{cases}
$$
